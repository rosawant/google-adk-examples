# Google ADK



## limitations:
1. Agent directory name and the agent name must be same
2. Agent name must be a valid identifier. It should start with a letter (a-z, A-Z) or an underscore (_), and can only contain letters, digits (0-9), and underscores
3. Google prebuilt tools cant be combined with the custom tools
4. Use models which supports tools [Ollama Tool Supporting Models](https://ollama.com/search?c=tools)
